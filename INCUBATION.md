# Incubation & Strategic Support

Hermes Orchestrator / wasm-agent is open to incubation, accelerators, grants, strategic funding, infrastructure credits, and pilot partnerships.

## What We Are Building

Hermes Orchestrator is a local-first control plane for agent fleets, plugins, browser-backed workspaces, and shareable agentic artifacts.

wasm-agent extends this direction into a spatial runtime where users can create, automate, fork, and share living spaces where agents, tools, browsers, devices, and workflows operate together.

The long-term goal is to make agent workspaces portable, inspectable, automatable, and shareable.

## Why Now

AI agents are moving from chat into operating environments.

The missing layer is a trustworthy runtime where agents can:

- observe workspace state
- control tools safely
- coordinate browser and device workflows
- operate across plugins
- package work into shareable artifacts
- support local-first use while connecting to hosted cloud resources when needed

Hermes Orchestrator / wasm-agent explores that layer.
