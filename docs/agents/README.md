# Agent Docs

`docs/agents` contains versioned documentation for Hermes Orchestrator node
profiles and runtime contracts.

## Current Docs

- `node.env.md`: bootstrap contract for `/local/agents/envs/<node>.env`.

Keep these docs aligned with `agents/README.md`, `agents/envs/README.md`, and
the current `horc` bootstrap code. If an environment variable changes behavior,
update all affected docs in the same change.
