# Wiki Engine Native Plugin

`wiki-engine` is the Hermes-native packaging for the shared wiki bootstrap,
repair, and query flow currently sourced from the legacy `hermes-core` plugin.

## Env Contract

- `PLUGIN_WIKI_ENGINE=true|false`

`PLUGIN_WIKI_ENGINE=true` is the intended enable flag for this package.

## Current Status

This package is currently a native-plugin scaffold for a future Hermes-native
wiki implementation.
