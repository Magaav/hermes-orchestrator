A bridge URL is an HTTP endpoint that lets wasm-agent talk to your own running Hermes-compatible agent.

                
Required routes: GET /health, GET /bridge/v1/models or /v1/models, and POST /bridge/v1/chat, /tasks, or a Hermes /v1/runs API.

                
Loopback and private LAN bridges are checked from this browser and must allow the wasm-agent origin with CORS.
