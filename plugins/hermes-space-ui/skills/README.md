# Hermes Space UI Skills

`skills/` contains Hermes Agent skill instructions shipped by the
`hermes-space-ui` plugin.

## Current Skill

- `space-ui/SKILL.md`: guidance for working with Space Agent UI/customware
  through the Hermes Space UI bridge.

Skill docs must reflect current bridge and module behavior. Future Space OS or
WASM browser goals should link to the Space OS roadmap until implemented.
